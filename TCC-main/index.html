Sobre o Projeto

Como Tudo Começou

Nosso projeto nasceu da paixão por aeromodelismo e da curiosidade em entender como a tecnologia poderia melhorar o desempenho de um avião de pequeno porte. Observando as limitações dos modelos convencionais, decidimos criar uma solução mais inteligente e acessível.

Utilizando a plataforma Arduino, desenvolvemos um sistema de controle que permite maior precisão nos movimentos, estabilidade automática e a possibilidade de coleta de dados em tempo real durante os voos.

Objetivos Principais

  • Criar um aeromodelo controlado por Arduino
  • Fazer um projeto acessível aos custos
  • Desenvolver telemetria para coleta de dados de voo
  • Construção de um site para a documentação

Processo de Montagem

1. Planejamento e Design

Começamos com extensa pesquisa sobre aerodinâmica e dimensionamento de aeromodelos. Utilizamos software CAD para projetar a estrutura e simular o comportamento aerodinâmico.



SketchUp AutoCAD

2. Seleção de Componentes

Escolhemos cuidadosamente cada componente: Arduino como controlador principal, Modulo NRF24L01+ de alta eficiência, Regulador de tensão 3.3 V, Capacitor 1000uf e 100uf, Indutor com nucleo de ferrite, entre outros componentes.

Arduino ProMini NRF24L01+ Regulador Capacitores

3. Construção da Estrutura

Utilizamos bandejas de deprom construir a estrutura leve porém resistente. Além de utilizar palitos de picolé, canudos e clips metálicos.

4. Montagem da Eletrônica

Integramos todos os componentes eletrônicos, juntamos a estrutura do avião com os componentes escolhidos, fazendo a montagem de delicada e garantindo que cada componente funcionaria corretamente.

5. Programação e Testes

Utilizamos o software IDE (Integrated Development Environment) do próprio Arduino para fazermos a programação e a conexão entre o avião e o controle.

// Exemplo de código ficticio para demonstração 
void computePID() {
    error = setPoint - input;
    integral += error * dt;
    derivative = (error - prevError) / dt;
    output = Kp*error + Ki*integral + Kd*derivative;
    prevError = error;
}

6. Voos de Teste

Realizamos diversos voos de teste, ajustando parâmetros e refinando o sistema. Coletamos dados de telemetria para análise pós-voo e melhoria contínua do projeto.

Dados Coletados

  • Velocidade do ar
  • Temperatura Ambiente
  • Altura e distancia que suporta
  • Consumo de bateria
